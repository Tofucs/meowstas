1. Download the project
2. Open terminal in the root folder of meowstas
3. If necessary install the OCaml Graphics library by running [opam install
graphics] and install the font by running [sudo apt-get install xfonts-base]
4. Run [dune exec meowstas] to start the game
5. It will show something like this 

[A wild Bulbasaur appears!           
1: Thunder Shock
2: Quick Attack
Choose a move (type number):]

6. Inputting 1 will choose attack 1, and 2 chooses attack 2
7. Play until either you or the AI pokemon dies, then the game is over and you have to run exec again
